# Runeshape Combinations Checker for Path of Exile 2

Удобный внутриигровой инструмент для автоматического распознавания рунических комбинаций (Runeshapes) в **Path of Exile 2**, расчета их рыночной стоимости на основе актуальной базы данных с **poe.ninja** и вывода цен в кастомном графическом оверлее прямо поверх клиента игры.

Интерфейс приложения полностью стилизован под каноничный визуальный стиль интерфейса Path of Exile 2 (темные тона, бронзовые рамки, золотые шрифты) и аккуратно встраивается в системный трей Windows в виде компактного флайаута.

---

## ✨ Основные возможности

* **Умное и экономичное сканирование:** Приложение не нагружает процессор постоянным распознаванием текста (OCR). Вместо этого оно использует легковесный мониторинг изменения яркости экрана, чтобы мгновенно определять, когда игрок открывает или закрывает интерфейс рунических комбинаций в игре.
* **Высокоточное распознавание текста:** Двухпроходной движок на базе **Tesseract OCR** с поддержкой мультиязычности и алгоритмами нечеткого поиска (**Fuzzy Matching / Расстояние Левенштейна**) для безошибочного сопоставления названий предметов, даже если текст частично перекрыт игровыми эффектами.
* **Синхронизация с poe.ninja:** Автоматическое фоновое обновление цен на валюту и предметы для выбранной лиги в режиме реального времени.
* **Кастомный игровой оверлей:** Цены отображаются в виде компактного полупрозрачного списка прямо поверх игрового окна строго рядом с нужными элементами.
* **Гибкое управление из трея:** Окно настроек скрывается в системный трей, поддерживает разворачивание по клику / двойному клику и автоматически привязывается к правому нижнему углу экрана (над панелью задач Windows).
* **Глобальные хоткеи:** Мгновенный старт и остановка сканирования по нажатию одной клавиши из любой точки операционной системы с возможностью переназначения хоткея «на лету».

## 🛠 Архитектура и принцип работы

Работа программы разделена на несколько изолированных и оптимизированных модулей:

1. **Конфигурация (`config.py`):** Хранит настройки выбранной лиги, сохраненные координаты калибровки экрана и глобальный хоткей.
2. **Загрузка цен (`price_repository.py`):** Асинхронно скачивает актуальные данные с API poe.ninja, парсит информацию о стоимости рун/комбинаций и запускает фоновый таймер для их периодического обновления.
3. **Мониторинг экрана (`scan_engine.py`):** * С частотой ~5 раз в секунду проверяет среднюю яркость в откалиброванной области экрана (алгоритм гистерезиса).
   * Если яркость соответствует открытому окну комбинаций, запускает OCR-сканирование (`ocr_scanner.py`) с частотой не чаще 1 раза в секунду.
   * Переводит распознанные названия (RU -> EN) через словарь локализации (`ru_translator.py`).
   * Ищет совпадения в локальной базе (`Runeshape_Combinations.json`) и сопоставляет их с ценами из репозитория.
4. **Интерфейс и Оверлей (`settings_window.py` / `overlay.py`):** Отрисовывает графические элементы с помощью **PySide6** (Qt) с использованием аппаратного ускорения, кастомных стилей StyleSheet и векторной графики QPainter.

## 🚀 Инструкция по установке и запуску

### 1. Требования к окружению
* Операционная система: **Windows 10 / 11**
* **Python 3.10** или выше
* Установленный в систему **Tesseract OCR**

### 2. Установка Tesseract OCR
1. Скачайте установщик Tesseract для Windows (например, от [UB Mannheim](https://github.com/UB-Mannheim/tesseract/wiki)).
2. При установке обязательно отметьте галочками установку дополнительных языковых пакетов (особенно **Russian** и **English**).
3. Убедитесь, что папка программы содержит директорию `tessdata` с файлами `rus.traineddata` и `eng.traineddata`.

### 3. Клонирование и установка зависимостей
Откройте терминал в папке с проектом и выполните:

```bash
# Установка необходимых библиотек
pip install -r requirements.txt

Если файла requirements.txt еще нет, базовые зависимости можно установить вручную:
Bash

pip install PySide6 pynput pillow rapidfuzz requests

4. Запуск приложения
Bash

python settings_window.py

🎮 Как пользоваться инструментарием

    Выбор лиги: При первом запуске в выпадающем списке окна настроек выберите актуальную лигу (например, Standard или текущую временную лигу). Программа автоматически начнет загрузку цен.

    Калибровка области:

        Нажмите кнопку "Калибровать область".

        На экране появится полупрозрачное окно выделения. Растяните его точно поверх той области игрового экрана, где в интерфейсе Path of Exile 2 отображаются рунические комбинации.

        Подтвердите калибровку. Координаты автоматически сохранятся в конфигурационный файл.

    Запуск трекера:

        Нажмите кнопку "Старт" в интерфейсе или воспользуйтесь глобальным хоткеем (по умолчанию настроен в конфигурации). Окно настроек автоматически свернется в трей.

        Теперь при открытии соответствующей панели в игре оверлей автоматически считает текст с экрана и выведет стоимость предметов в хаосах или дивайн-орбах.

    Изменение хоткея: Если дефолтная клавиша неудобна, нажмите кнопку "Изменить" в блоке хоткея, нажмите любую желаемую клавишу на клавиатуре, и новый бинд тут же вступит в силу.
